This plugin allows to create easy display notification message popup that informs site user to use cookies for your website.

== Description ==

= It Displays a Cookie Notification in customized Popup on your WordPress site =

This plugin displays a customized popup notification message to your site's visitors for your WordPress site to use cookies. Whether you want to comply with the EU Cookie Law or just to help your visitors out, this is a simple plugin with minimal impact to your site's performance. This plugin allows to customize popup theme also with color and position changes. 

= Quick Note =

Cookie lifetime
Set cookie life time for 15 days by default. Admin can change it as per requirement. After the preset days of cookie expires, cookie will be auto deleted. Once cookie(Name 'aws_visited') is deleted, again notification popup will be displayed when we hit website for first time.

Support provided to ckeditor plugin for 'Cookie Notification Message' field to easy user interface.

== Installation ==

AWS Cookies Popup is simple to install:

1. Unzip aws-cookie-popup.zip
2. Upload 'aws-cookie-popup' directory to your '/wp-content/plugins' directory
3. Go to the plugin management page and enable the AWS Cookies Popup.
4. Browse to Settings > Cookie Popup
5. Customise your cookie notification message with position and color for your WordPress site.


== Frequently Asked Questions ==

= This plugin use cookie to store data? =

Yes, This plugin is used 'aws_visited' cookie to store the data.

= Can I change cookie lifetime? =

By default this plugin takes 15(Days) as cookie lifetime. You can change it from Settings > AWS Cookie Popup > Cookie Expire. Make sure that value is entered in day.
